* Open Test Robot Platform

** Prerequisites

*** Mandatory

- Python 2.7 or later.

*** Recommended

- fMBT for software emulation of robot and camera. (Available from
  https://github.com/01org/fMBT.)

- Sphinx for building documentation (python-sphinx package in Debian/Ubuntu).
